## Preguntas para validar el avance

### 📋 Preguntas metodológicas

| # | Pregunta | Por qué preguntar | Opciones posibles |
|---|---|---|---|
| **1** | ¿Acepta comparar censos de distintos años (2010 vs 2017)? | Los censos no son simultáneos (Chile 2017, Brasil 2010, Argentina 2010, Perú 2017). ¿Cree válido el comparativo temporal? | - Sí, es válido<br>- Solo mismo año<br>- Solo mismo país<br>- Depende del tema |
| **2** | ¿Prefiere ver conteos absolutos, porcentajes o ambas? | Determinar qué tipo de resultados priorizar en Power BI. | - Conteos absolutos<br>- Porcentajes<br>- Ambas con equilibrio |
| **3** | ¿El tamaño poblacional debe agregarse por comuna/municipio/departamento/distrito o prefiere otro nivel? | Estándardizar la unidad geográfica para comparar tamaño entre países. | - Comuna (Chile)<br>- Municipio (Brasil)<br>- Departamento (Argentina)<br>- Distrito (Perú)<br>- Agregado único |
| **4** | ¿Para GeoDa en Chile, prefiere usar comuna o región como unidad espacial? | GeoDa requiere datos por unidad espacial definida. | - Comuna<br>- Región<br>- Ambos escalonados<br>- Otro nivel |
| **5** | ¿Acepta indicadores aproximados (no variables idénticas exactas) para comparar países? | Las variables pueden no tener nombres idénticos entre países. | - Sí, equivalentes funcionales<br>- No, mismas variables exactas<br>- Depende de la fiabilidad |

## Notas para el estudiante

### Antes de preguntar:

- Tenga el Excel abierto para mostrar ejemplos concretos
- Identifique qué preguntas son realmente necesarias (3-5 clave)
- Prepare ejemplos visuales si aplica

### Durante la reunión:

- Escuche activamente las respuestas
- Tome notas sobre preferencias y restricciones
- Agradezca el feedback recibido

### Después de la reunión:

- Documente las respuestas obtenidas
- Ajuste el plan según feedback
- Comunique cambios al equipo si aplica
